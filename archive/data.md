### Pattern Breakdown by Dataset Type:
1. Complementary (20_complementary.csv)
Doc A: First piece of complementary information
Doc B: Second piece of complementary information
Docs C-J: Background distractors
2. Duplicate (20_duplicate.csv)
Doc A: Information stated one way
Doc B: Same information paraphrased/restated
Docs C-J: Background distractors
3. Synergy (20_synergy.csv)
Doc A: Identifies the entity or provides context
Doc B: Describes the key property/mechanism
Docs C-J: Background distractors
